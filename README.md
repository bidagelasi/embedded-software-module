# gpio模拟串口TX
该模块用MCU的一个普通gpio口实现了模拟串口TX的功能。该功能的用处在于在一些MCU缺乏硬件串口的情况下，用一个gpio来模拟串口TX，以输出调试信息或者把一些数据发送到对端MCU的串口接收端。
